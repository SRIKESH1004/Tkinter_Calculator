# Calculator (Tkinter)

File: [calculator.py](calculator.py)

Description
- Simple GUI calculator using Tkinter.
- Main functions:
  - [`press`](calculator.py) — insert a value into the entry.
  - [`clear`](calculator.py) — clear the entry.
  - [`cal`](calculator.py) — evaluate the expression and display result.

Prerequisites
- Python 3.x
- Tkinter (usually included with Python; on Debian/Ubuntu: `sudo apt install python3-tk`)
